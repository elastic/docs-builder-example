---
title: RST in Markdown
---

You can use the `eval-rst` directive to include ReStructuredText in a markdown file.

```{eval-rst}
.. code-block:: yaml

    project:
    title: MyST Markdown
    github: https://github.com/jupyter-book/mystmd
    license:
        code: MIT
        content: CC-BY-4.0
    subject: MyST Markdown
```
